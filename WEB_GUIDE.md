# Web界面使用说明

## 启动Web服务器

```bash
npm run web
```

服务器将在 `http://localhost:3000` 启动。

## 功能介绍

### 1. 主页面
访问 `http://localhost:3000` 可以看到：
- 当前日期和时间（实时更新）
- 百度热搜和微博热搜切换标签
- 20条实时热搜数据（卡片式展示）
- 刷新按钮

### 2. 切换数据源
- 点击顶部的标签切换：
  - **百度热搜** - 稳定可用
  - **微博热搜** - 可能受限

### 3. 查看详情
- 点击任意新闻卡片
- 弹出详情窗口显示：
  - 新闻标题
  - 排名
  - 热度值（格式化显示）
  - 数据来源
  - 查看完整内容链接

### 4. 刷新数据
- 点击顶部"刷新数据"按钮
- 重新获取最新的热搜信息

## 快捷键

- **ESC** - 关闭详情弹窗
- **点击弹窗外部** - 关闭详情弹窗

## API端点说明

### GET /api/baidu-hot
获取百度热搜数据

**响应示例：**
```json
{
  "success": true,
  "data": [
    {
      "rank": 1,
      "title": "新闻标题",
      "url": "https://...",
      "hot": "1234567"
    }
  ],
  "timestamp": "2026-01-02T12:00:00.000Z"
}
```

### GET /api/weibo-hot
获取微博热搜数据

**响应示例：**
```json
{
  "success": true,
  "data": [
    {
      "rank": 1,
      "title": "话题标题",
      "url": "https://...",
      "hot": "1234567",
      "category": "娱乐"
    }
  ],
  "timestamp": "2026-01-02T12:00:00.000Z"
}
```

## 技术栈

- **后端**: Express.js
- **前端**: 纯HTML + CSS + JavaScript
- **API**: RESTful接口
- **样式**: 渐变色 + 卡片式设计
- **响应式**: 支持手机、平板、桌面

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
- Edge

## 注意事项

1. Web服务器占用端口 3000
2. 确保端口未被其他应用占用
3. 如需修改端口，编辑 `server.js` 中的 `PORT` 变量
4. 微博热搜可能受API限制

## 停止服务器

在终端按 `Ctrl + C` 停止服务器。
